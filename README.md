# BibTeX Filter App

A simple web app to filter and clean BibTeX entries. You can select which fields to keep and generate a clean BibTeX entry ready to use in your `.bib` file.

---

## How to Use

1. Follow the URL below:
[https://bibtex-filter-app-4xyvxcbftq3nouunxt25fz.streamlit.app/](https://bibtex-filter-app-4xyvxcbftq3nouunxt25fz.streamlit.app/)

2. Paste your BibTeX entry in the text area.  

3. Select the fields you want to include.  

4. Click **"Generate Filtered BibTeX"** to get your clean entry.

---

## Features

- Select fields such as `title`, `author`, `journal`, `year`, and more.  
- Removes extra spaces and formats BibTeX correctly.  
- Works for single entries, with support for multiple entries in future updates.

---

## Tech Stack

- Python  
- [Streamlit](https://streamlit.io) for the web interface
